## 你好，世界！

这是我个人博客的第一篇文章。

博客使用 [Hugo](https://gohugo.io/) 静态站点生成器搭建，主题为 [LoveIt](https://github.com/dillonzq/LoveIt)，通过 GitHub Actions 自动部署到 GitHub Pages。

### 技术栈

- **Hugo** — 基于 Go 的静态站点生成器，构建速度极快
- **LoveIt** — 功能丰富、简洁美观的 Hugo 主题
- **GitHub Pages** — 免费的静态网站托管
- **GitHub Actions** — 自动化构建和部署

### 写作流程

1. 在 `content/posts/` 目录下创建 Markdown 文件
2. 编写文章内容
3. `git push` 推送到 GitHub
4. GitHub Actions 自动构建并部署

就是这么简单！开始记录吧。
